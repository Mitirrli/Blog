K8S 节点状态无法调度,状态为 SchedulingDisabled

解决方案(*为 ip):

1

$ kubectl patch no * -p '{"spec": {"unschedulable": false}}'

Back-off restarting failed container

根据 Describe Pod,pod 内的 Container 已完成.但是 Pod 的生命周期非常短,要使 Pod 持续运行,您必须指定一个永远不会完成的任务.
